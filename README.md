# GenAI-Labs: Laboratório de Inteligência Artificial Generativa

Este repositório é dedicado ao estudo e à aplicação prática de modelos de linguagem (LLMs) e frameworks de IA Generativa. O objetivo é construir um portfólio de projetos que demonstrem a capacidade de integração de modelos em soluções de software.

## Foco de Estudo

* **LLMs:** Google Gemini, OpenAI (GPT).
* **Frameworks:** LangChain (para orquestração de prompts e ferramentas).
* **Conceitos:** Prompt Engineering, RAG (Retrieval-Augmented Generation), e Fine-Tuning básico.
